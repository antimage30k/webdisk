## WebDisk
### 后端
#### 基本功能
- 存储图片和文件
- 文件私享和分享功能
- 文件管理
#### 实现
- 本地文件夹存储，分文件夹，uuid重命名，mysql管理，
- 用户系统，token实现私享
- md5防重，新增、下载、列表、删除功能

### 前端
#### 基本功能
- 展示图片
- 文件上传
- 前后端部分分离

### 架构
- 后端框架：Flask
- ORM：sqlalchemy
- 前端模版：Jinja2
- 进程管理：supervisor
- 反向代理：Nginx

### 特性
- 文件去重，md5 验证文件是否已存在
- 文件公开等级分为私有、公开和认证（即登录用户可以看到）
